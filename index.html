Welcome to the Thaumatorium!

This is my handcrafted website, where I write articles and post my projects.

Note that this site is pure uncompressed HTML/CSS/JS, with some PNG and WEBP images to fluff thing up. I like to think that this is how the web is supposed to be. This site is (for now) a collection of static pages instead of built up by some (C#/JS/PHP/Java) framework.

Everything is handcrafted and not minified, something that's usually the case, so if you want to figure out how something works, you can just go and check it out - right click on something and then click "Inspect".

If you want to know more about this site, check out the About page.
